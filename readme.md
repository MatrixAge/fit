## Motivation

因为早年为了获取技术经验而996甚至007，长期的久坐和熬夜，导致我迅速肥胖。毕业时六十多KG，在学校时每星期至少有4天都有进行长跑，每次绕着大操场外圈跑步40圈，而毕业之后因为作息原因，就很少运动了，当一个人运动量陡然下降的时候，身体会慢慢膨胀，到现在，我已经150kg了，肚子上也多了一层肉，最关键的是，长期沉浸在快乐之中，人的意志力会慢慢衰退，精气神会消耗殆尽，人就有一股老气，这是我不能接受的。

在进行一段时间的健身之后，我试用了市面上国内外的很多健身应用，全部，我是说全部，都无法满足我的需求。我的要求很简单：拥有计时器功能，并能自定义分组。但市面上的App都做的太烂了，基本都沦为了售卖健身课的平台。

我很失望，最好用的竟然是手机自带的闹钟，尽管我只使用里面的计时功能。偶然在B站up主Eric的跳绳视频中看到他使用的一个App，感觉那就是我要的，打听之后发现，该App下架了。

所以就有了Fit，做商业开源应用，我一直都有这个想法，之前一直在纠结，你做了没有人用怎么办，后来想通了，做了就有价值，just do it！我做这个的初衷是自己用，把自己的身体搞好了，钱什么的，慢慢去赚，人生长着呢。
